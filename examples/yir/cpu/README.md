# CPU `YIR` Examples

This folder contains CPU-focused handwritten `YIR` examples:

* arithmetic and bitmix
* typed values and structs
* heap/buffer ownership demos
* small CPU-only driver paths
* async-first handwritten shorthand via [cpu_async_shorthand.yir](/Users/Shared/chroot/dev/nuislang/examples/yir/cpu/cpu_async_shorthand.yir)

Recommended starting points:

* [cpu_types_struct.yir](/Users/Shared/chroot/dev/nuislang/examples/yir/cpu/cpu_types_struct.yir)
  typed scalar + struct surface
* [cpu_linked_list_rustish.yir](/Users/Shared/chroot/dev/nuislang/examples/yir/cpu/cpu_linked_list_rustish.yir)
  ownership-flavored linked-list path
* [cpu_async_shorthand.yir](/Users/Shared/chroot/dev/nuislang/examples/yir/cpu/cpu_async_shorthand.yir)
  compact handwritten async/lane style
